## 📖 About

KeenKeeper is a friend relationship management app built with React. It helps you keep track of your meaningful connections, log interactions, and visualize your communication patterns. Never lose touch with the people who matter most.

---

## 🛠️ Technologies Used

- **React 19** — Frontend UI library
- **React Router v7** — Client-side routing with loaders
- **Tailwind CSS v4** — Utility-first styling
- **DaisyUI** — Pre-built UI components
- **Recharts** — Data visualization (Pie Chart)
- **React Toastify** — Toast notifications
- **React Icons** — Icon library
- **Context API** — Global state management

---

## ✨ 3 Key Features

### 1. 📋 Friend Dashboard
Browse all your friends at a glance with detailed profile cards showing contact status, tags, and days since last interaction. Summary cards provide instant statistics including total friends, on-track connections, and those needing attention.

### 2. 📞 Quick Check-In with Timeline Tracking
Log interactions (Call, Text, Video) with any friend through a single click. Every interaction is recorded in a dedicated Timeline page with filtering options to view specific interaction types. Toast notifications provide instant feedback for every action.

### 3. 📊 Friendship Analytics
Visualize your communication patterns with an interactive donut chart powered by Recharts. See the breakdown of your interactions by type (Call, Text, Video) to understand how you stay connected with your friends.
